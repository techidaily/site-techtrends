## How to Download and Install VirtualDub on Mac with Wine

Wine is open source software that helps run Windows applications on other OS like Mac and Linux. PlayOnMac, Wineskin, WineBottle and CrossOver are wrappers of Wine. You can free download VirtualDub for Mac and install it via the listed software, among which PlayOnMac is free and easy to use. We'll walk you through the details on how to make VirtualDub run on Mac and let it work as a video converter for Mac via PlayOnMac. 

1. Download PlayOnMac (the latest version is 4.2.10) and install it on your computer. Then free download VirtualDub for Mac as usual.
2. Run PlayOnMac, select Install > Install an application not included in the list > Install the application in a new virtual drive > Install some library > POL\_Install\_vcrun6 > Explorer and choose VirtualDub Mac installer you downloaded and install VirtualDub on Mac.
3. Head to Finder, drag VirtualDub for Mac into folder in /User/your\_username/PlayOnMac's virtual drives/your\_drive\_name/drive\_c/Program Files.
4. In PlayOnMac, click "Configure" and select the virtual drive you created.
5. Choose "Create a new link for this virtual drive" under "General" section and select "VirtualDub.exe" and go to the next step to install VirtualDub for Mac.
6. Enter a name and click Next. Now, you'll see the app.

### Step 1: Load .dvdmedia File

To convert .dvdmedia to MP4, you need to change your .dvdmedia file to a VIDEO\_TS folder first. It's easy. Just follow the steps below. 

1. Right-click the .dvdmedia file and select "Get Info".
2. Delete the (.dvdmedia) extension from both the name and extension fields.
3. Click "Remove" in the pop-up window to transform the DVDMEDIA file into a VIDEO\_TS folder.

Then, open MacX DVD Ripper and click the + Folder button to load the VIDEO\_TS folder that is converted from your DVDMEDIA file.

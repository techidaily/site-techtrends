## 2\. Transfer DVD Movies to DLNA Media Server (Plex, NAS, PlayOn etc.)

Once you got your DVD library ripped to digital formats, you're now able to transfer DVD movies to media server. But first you need to get a DLNA media server. There are a number of media servers available which will do this either free or economically.

### Pick a Proper DLNA Media Server to Store DVD Movies

**1\. Plex (Windows/Mac/Linux)**

Plex is a well-known media server running on cross-platform, with a wide range of supported devices and easy-of setup. It's free to share media on your local network, with Plex offering a paid Plex Pass to enable additional features such as cloud streaming and media syncing. 

**Pros:** Painless setup; Supports transcoding to devices; Metadata processed on server  
**Cons:** Parts are closed-source; Mobile apps and Plex Pass are paid options 

![Plex media server](https://www.macxdvd.com/mac-dvd-video-converter-how-to/article-image/transfer-dvd-to-media-server-2.jpg) 

**2\. Universal Media Server (Windows/Mac/Linux)**

Universal Media Server is a media server capable of serving videos, audio and images to any DLNA-capable device. It is free, regularly updated and has more features than any other media server, including paid media servers. It streams to many devices including PS3, PS4, Microsoft Xbox One and 360, many TVs (Samsung, Panasonic, Sony, Vizio, LG, Philips, Sharp), smart phones (iPhone, Android, etc.), Blu-ray players, and more.

**Pros:** Open-source; Cross-platform; Free  
**Cons:** Uses Java

![Universal media server](https://www.macxdvd.com/mac-dvd-video-converter-how-to/article-image/transfer-dvd-to-media-server-3.jpg) 

**3\. PlayOn (Windows)**

PlayOn is a DLNA media server and browser for your Windows PC . It is one of the few media servers that offers support for online media sources such as Netflix, Hulu, Amazon, HBO and 100 other popular streaming sites. It makes it possible to watch anywhere, anytime, on any device, without the commercials, and even offline.

**Pros:** Allows easy integration of online media sources such as Netflix and Hulu; Supports local and online content  
**Cons:** Only available on Windows

![Playon Media server](https://www.macxdvd.com/mac-dvd-video-converter-how-to/article-image/transfer-dvd-to-media-server-4.jpg) 

## How to Stream DVD to Plex Media Server

In view of the fact that Plex media server is widely welcomed to [stream DVD movies to Apple TV](https://tools.techidaily.com/macxdvd/products/), Chromecast, Roku box, here I take Plex as an example to transfer DVD to media server.

**Step 1:** Sign up for your free account and download the Plex Media Server software, which will organizes all of your media including converted DVD digital copy.

**Step 2:** Add library by clicking "Add Section" button and selecting "Movies". Plex recognizes just about any kind of video format and displays all of your content in an easy-to-navigate interface.

**Step 3:** Select the DVD rips you want to add and click the Add Folder button. 

**Step 4:** Tune into Plex and by use of DLNA-friendly device, like [stream DVD to Roku](https://tools.techidaily.com/macxdvd/products/) box, Chromecast, gaming console for watching on different devices.
